Respond immediately when an agent breaches deterministic safety or security rules.

This capability handles hard breaches such as unauthorised tool use, identity violation, sandbox escape, data exfiltration attempt or prohibited action. It terminates or contains the agent and triggers compensation or rollback where required.

Purpose and strategic objective

Purpose

To provide zero-tolerance response for deterministic breaches that cannot be safely allowed to continue.

Strategic objective

Reduce blast radius by stopping unsafe AI activity before it creates further business, security or data impact.

Architectural placement

Why this sits here

This is part of the Decision Routing Gateway, not a layer. It receives hard-breach signals from runtime controls and routes them directly to containment and response.

Hard constraints should not be debated by the agent; they should be enforced.
Problem statement

Some agent behaviours are not matters of confidence or judgement. They are policy violations that require immediate action.

Capability description

What the capability does

  • Terminates or suspends unsafe agents or sessions.
  • Revokes credentials and access tokens.
  • Blocks tool, API or network paths.
  • Triggers rollback, compensation or recovery playbooks.
  • Records evidence for incident response.

What the capability does not do

  • It does not determine soft policy ambiguity.
  • It does not replace root cause analysis.
  • It does not remove the need for post-incident review.

Enterprise examples

Tool violation

Agent attempts to call an unapproved payment API.

Normal / desired state:
Execution stops and credentials are revoked.
Risk / control concern:
Agent continues making calls after violation.

Sandbox escape

Generated code attempts to access host filesystem.

Normal / desired state:
Sandbox is terminated and alert created.
Risk / control concern:
Code escapes and persists.
Failure scenarios

Delayed token revocation

Agent continues using active credential.

Partial transaction

Agent starts but does not complete a business action.

Missing evidence

Response team cannot reconstruct why containment occurred.
