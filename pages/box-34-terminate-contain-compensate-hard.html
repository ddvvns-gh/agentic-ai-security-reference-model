Enterprise scenarios

Runaway loop

Agent repeatedly calls API.

Desired state:
Kill switch and token revocation.
Concern:
Backlog and cost growth.

Unsafe message

Agent sends incorrect commitment.

Desired state:
Freeze and remediation workflow.
Concern:
Customer harm.

Poisoned memory

Agent acts on compromised memory.

Desired state:
Quarantine memory and contain agent.
Concern:
Poison persists.
