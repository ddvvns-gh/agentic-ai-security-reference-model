Box 30 Full Depth Upgrade v3

Agent-to-Agent Governance establishes semantic blast-radius isolation and transitive identity controls.

Control Objective

Isolate semantic blast radiuses across multi-agent systems and prevent downstream prompt injection from mutating higher-privilege contexts.

Control Statement

Cross-boundary agent communications must use strongly typed schemas, field-level validation and task-bound credentials. No downstream agent may pass unvalidated natural language into a higher privilege context.
