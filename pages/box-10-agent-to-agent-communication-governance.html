Agent-to-Agent Governance establishes semantic blast-radius isolation and transitive identity controls.

Control Objective

Isolate semantic blast radiuses across multi-agent systems and prevent downstream prompt injection from mutating higher-privilege contexts.

Control Statement

Cross-boundary agent communications must use strongly typed schemas, field-level validation and task-bound credentials. No downstream agent may pass unvalidated natural language into a higher privilege context.

Risk Catalogue

  • R-A2A-001 Semantic contamination
  • R-A2A-002 Cross-agent privilege escalation
  • R-A2A-006 Transitive identity persistence

Threat Events

  • Child agent injects manipulated context into parent workflow
  • Compromised child token survives parent task completion
  • Credential reused for lateral movement

Controls

  • C-A2A-001 Schema-enforced payload exchange
  • C-A2A-005 Downstream Privilege Ceiling Policy
  • C-A2A-006 Task-Bound Credential Lifecycle
  • Semantic Blast Radius Isolation

Metrics

  • M-A2A-007 Payloads validated via schema = 100%
  • M-A2A-008 Semantic anomalies intercepted
  • M-A2A-009 Credential expiry compliance
  • M-A2A-010 Child sessions exceeding parent lifetime = 0

Evidence

  • E-A2A-003 Structured Payload Validation Records
  • E-A2A-004 Credential Lifecycle Audit Records
