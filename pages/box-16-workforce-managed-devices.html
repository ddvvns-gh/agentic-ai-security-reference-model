Control AI usage from enterprise-managed endpoints and browsers.

Managed devices provide the strongest endpoint control point for AI usage. This capability governs browser sessions, local execution, clipboard, downloads and device posture where the organisation has administrative control.

Purpose and strategic objective

Purpose

To ensure employee AI interactions on managed devices occur through approved, monitored and policy-enforced channels.

Strategic objective

Reduce data leakage, local execution and unmanaged AI access risks from corporate endpoints.

Architectural placement

Why this sits here

This sits in Layer 0 because managed devices are direct human entry points into AI systems and therefore part of the attack surface.

The first AI security boundary is often the employee browser or endpoint.
Problem statement

Users can copy sensitive data into external AI, execute generated code locally, or use browser plugins that bypass enterprise gateways.

Capability description

What the capability does

  • Enforces enterprise browser and endpoint policies.
  • Controls clipboard, downloads, uploads and local execution.
  • Supports DLP and session monitoring.
  • Restricts unmanaged AI sites and extensions.
  • Produces endpoint and browser evidence.

What the capability does not do

  • It does not replace runtime AI gateway controls.
  • It does not control unmanaged devices.
  • It does not make unsafe user intent safe by itself.

Enterprise examples

Employee uses Copilot-style tool

A managed laptop accesses approved AI.

Normal / desired state:
Session controls and DLP policies apply.
Risk / control concern:
User pastes sensitive data into an unmanaged AI site.

Developer executes generated script

AI produces a script for troubleshooting.

Normal / desired state:
Execution is restricted or routed to sandbox.
Risk / control concern:
Generated code runs with excessive local privilege.
Failure scenarios

Sensitive upload

A user uploads confidential files to an unapproved AI service.

Malicious generated code

Generated code modifies local credentials.

Extension exfiltration

An AI browser extension captures prompts and page content.
