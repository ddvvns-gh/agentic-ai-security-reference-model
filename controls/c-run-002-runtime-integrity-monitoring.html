C-RUN-002

Runtime Integrity Monitoring

Monitor admitted workloads at run time for drift and tampering — so a workload that was safe at admission cannot silently change after it starts.

Control IDC-RUN-002
TypeDetective
Trust domainExecution
Enforcement pointRuntime / control plane
Failure postureAlert
Parent capabilityBox 26 · Compute Admission & Runtime Integrity
Control objective
Close the gap between admission and execution. A workload can be tampered with or drift after it is admitted; runtime integrity monitoring detects unexpected changes in the running process and triggers response.
Control statement
Admitted workloads must be monitored at run time for integrity drift (unexpected binaries, configuration or behaviour); significant drift must raise a finding and may trigger containment (Box 34).
Design & operation
  1. Baseline the admitted workload's expected state.
  2. Monitor the running workload for drift (unexpected processes, file/config changes, behaviour).
  3. Significant drift raises a finding and may trigger containment (Box 34).
  4. Drift events recorded; measured (M-RUN-002).
Inputs / outputs

Inputs

  • Admitted-workload baseline
  • Runtime telemetry

Outputs

  • Drift findings
  • Containment triggers (Box 34)
  • Drift events (M-RUN-002)

Test method / assurance

Design effectiveness

  • Confirm baselines exist for admitted workloads.
  • Confirm drift detection covers process/file/behaviour.
  • Confirm drift can trigger containment.

Operating effectiveness

  • Tamper with a running workload; confirm detection.
  • Measure drift events (M-RUN-002).
  • Confirm severe drift triggers containment.
Failure modes & residual risk
  • Subtle drift. Low-and-slow changes may evade; tighten baselines.
  • Residual. Medium — detective.
Regulatory mapping
  • APRA CPS 234 — integrity monitoring.
  • NIST AI RMF — Manage.
  • MITRE ATLAS.
Implementation notes (vendor-neutral)
Implement runtime integrity/behaviour monitoring (eBPF/agent-based) against the admitted baseline; wire severe drift to containment (Box 34).
