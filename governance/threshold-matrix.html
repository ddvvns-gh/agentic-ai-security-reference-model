How a classified action is routed

The Enterprise Commitment Threshold Matrix classifies an action by impact (T0–T3) and certainty (hard / soft). This page is the routing rule the gateway applies to that classification — impact-first precedence into terminate, evaluate or escalate.

Precedence (evaluated top to bottom)
If…RouteGateway action
Impact ≥ T2 — regardless of certaintyHigh-impact → Pause & Escalate [39]Freeze; require owner / dual-control / quorum / HITL; transaction signing at T3.
else Hard breach (deterministic)Terminate / Contain [34]Revoke, terminate, isolate, compensate or roll back by reversibility & dependency map.
else Soft deviation (ambiguous)Evaluate / Yield [36/37]Allow/deny the specific action; record drift; policy-boundary change needs an approved update.
