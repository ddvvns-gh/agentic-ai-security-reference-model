# Executive Landing Page Update

This package changes the site entry model:

- `index.html` is now an executive-oriented landing page.
- `architecture.html` is the generic architecture view.
  - Palo Alto / Idira overlay controls removed.
  - Gap / Existing / Maturing maturity badges removed from capability boxes.
- `theindex.html` preserves the previous working architecture view with overlays and maturity indicators.
- Supporting folders and capability pages are preserved from the v18 package.

Recommended repository update:
1. Back up current repository.
2. Copy all files from this package into the repository root.
3. Commit and push.
4. Visit:
   - `/` for the executive entry page.
   - `/architecture.html` for the generic architecture view.
   - `/theindex.html` for the working view.
