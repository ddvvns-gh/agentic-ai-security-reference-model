R-COST-003

Cost Control Bypass

Cost controls are circumvented — spend flows through un-metered or direct paths that the budget and limits never see.

Risk IDR-COST-003
CategoryOperational Detection & Resilience
Trust domainExecution
ReversibilityRecoverable / availability
OwnerSOC / Ops
Parent capabilityBox 31 · FinOps & Runtime Cost Controls
Description
Budgets and loop limits only bind consumption that passes through the metered control plane. Direct provider calls, un-brokered tools or alternate keys create un-metered paths where runaway cost can occur unchecked.
Threat scenario
An agent (or its tooling) calls a model provider directly with a separate key, outside the AI gateway. The runtime budget never meters it, so a runaway there is invisible until the provider invoice arrives.
Assessment
LikelihoodMediumUn-brokered/direct paths are common in immature estates.
ImpactHighUnbounded, invisible spend; defeats the cost controls.
InherentHighBefore controls.
