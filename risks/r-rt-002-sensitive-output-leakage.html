R-RT-002

Sensitive Output Leakage

An agent's output discloses sensitive data — personal, confidential or regulated — to a party that should not receive it.

Risk IDR-RT-002
CategoryRuntime / Output
Trust domainExecution
ReversibilityIrreversible (disclosure)
OwnerSecurity · Privacy
Parent capabilityBox 28 · Prompt & Output Inspection
Description
Agents synthesise across data they can access and may surface sensitive content in an output, intentionally (exfiltration) or incidentally (over-sharing in a helpful answer). Disclosure is irreversible once sent.
Threat scenario
An agent answering a customer query inadvertently includes another customer's details pulled from retrieved context, sending personal data to the wrong recipient.
Assessment
LikelihoodMediumOver-sharing and exfiltration are both common.
ImpactHighPrivacy breach; irreversible disclosure.
InherentHighBefore controls.
